Hacks avec CMD


Voici quelques actuces avec CMD !!!


Actuce N'1 : Faire semblant de Hacker (Prank) 😂

Vous Voulez faire un prank à vos amis? Essayez celui là !

Étape 1: Ouvrez CMD. Recherchez simplement CMD dans votre ordinateur ou faites Windows+R et tapez CMD (cmd).

Étape 2: Tapez color 2 pour modifier la couleur en vert comme un Vrai hacker !

Étape 3: Tapez tree c:/ et apelez vos amis ! Ils seront choqués !!! 😱😱😱

Actuce N'2 : Trouver la clé d'un réseau 🔑📶

Vous cherchez la clé de votre réseau ? Avec cette actuce vous pouvez trouver la clé de n'importe quel réseau ayant une clé !!!

Étape 1: Ouvrez CMD (Voir étape précédente)

Étape 2: Tapez netsh Puis Enter.

Étape 3: Tapez wlan show profiles pour voir tout les réseaux.

Étape 4: Chercher le reseau auquel vous voulez trouver la clé.

Étape 5: Tapez wlan show profiles *nom du site*.

Étape 6: Tapez wlan show profiles *nom du site* key=clear.

Étape 7: Descendez et cherchez contenu de la clé et vous trouverais le code.

Actuce N'3 : Pirater un site web 👨‍💻

Je vais vous montrer comment entrer dans le réseau d'un site pas sécurisé et supprimer des fichiers. ATTENTION : NE PAS UTILISER À USAGE ILLAGAL

Étape 1: Tapez ping *nom du site*(ex: www.hackertype.com) pour trouver l'IP

Étape 2: Tapez ftp -n.

Étape 3: Tapez open.

Étape 4: Tapez *nom du site*.

Étape 5: Attendez que ça se connecte.

Étape 6: Puis tapez un point d'interrogation(?).

Étape 7: Ça montre les commandes du site.

Étape 8: Pour supprimer tapez del après la commande(ex: ? del)

Actuce N'4 : Voir son IP 👍

Étape 1: Tapez dans le CMD ipconfig

Étape 2: Cherchez IpV4 et vous pouvez voir votre IP

Actuce N'5 : Éteindre un ordi à distance 😱

Étape 1: Tapez shutdown -i.

Étape 2: Faites "ajouter un ordinateur"

Étape 3: Tapez l'ip.

Étape 4: Mettez les paramètre que vous voulez et puis faites "ok"


Actuce N'6 : Rénitialiser COMPLETEMENT votre ordi !!! 😱

Étape 1: Tapez systemreset --factoryreset

Étape 2: Mettez les paramètre que vous voulez


Actuce N'7 : Libérer de l'espace dans votre ordi !!! 👍

Tapez del/q/f/s%temp%\* et faites "Enter" C'est tout !!!
